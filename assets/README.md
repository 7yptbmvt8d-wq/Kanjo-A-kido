# Assets

Les fichiers `.svg` présents ici sont des **placeholders** générés pour que le
site s'affiche correctement. Remplacez-les par les visuels officiels.

## Logos officiels à fournir

| Fichier attendu | Usage | Tailles de réf. |
|---|---|---|
| `kanji-kanjo.png` | Kanji « Kanjo » dorés, fond transparent — nav, hero, footer | 38px (nav), 158px (hero), 34px (footer) |
| `kanji-kanjo-white.png` | Même glyphe en blanc (fonds clairs) — optionnel | — |
| `logo-noir-1024.png` | Logo carré complet 1024×1024 — favicon, OG, Maps | 1024×1024 |

## Comment basculer sur les vrais PNG

Les pages référencent actuellement `assets/kanji-kanjo.svg` et
`assets/logo-square.svg`. Deux options :

1. **Renommer** vos PNG en `kanji-kanjo.svg` / `logo-square.svg` (déconseillé), ou
2. **Remplacer les chemins** : dans chaque `.html`, remplacer
   `assets/kanji-kanjo.svg` → `assets/kanji-kanjo.png` et
   `assets/logo-square.svg` → `assets/logo-noir-1024.png`.

Placeholders actuels :
- `kanji-kanjo.svg` — kanji 感情 dorés (vertical).
- `logo-square.svg` — logo carré noir & or (favicon / OG).
